# Portfolio Website

고경희의 개인 포트폴리오 웹사이트입니다. 모던하고 반응형 디자인으로 제작되었습니다.

## 📋 프로젝트 소개

이 포트폴리오 웹사이트는 HTML, CSS, 바닐라 자바스크립트만을 사용하여 구현되었습니다. PC와 모바일 환경 모두에서 최적화된 반응형 디자인을 제공합니다.

## ✨ 주요 기능

- **반응형 디자인**: 데스크톱, 태블릿, 모바일 모든 기기에서 최적화된 레이아웃
- **히어로 섹션**: 밤하늘 은하수 배경 이미지와 함께하는 인상적인 첫 화면
- **프로젝트 갤러리**: 3개의 프로젝트를 카드 형태로 소개
- **Contact 폼**: 방문자가 직접 연락할 수 있는 문의 폼
- **부드러운 스크롤**: 섹션 간 이동 시 부드러운 스크롤 애니메이션
- **모바일 네비게이션**: 모바일 환경에서 햄버거 메뉴로 네비게이션 접근

## 🛠️ 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: 
  - CSS Variables (커스텀 속성)
  - Flexbox & Grid 레이아웃
  - 반응형 미디어 쿼리
  - CSS 애니메이션 및 트랜지션
- **JavaScript (Vanilla)**: 
  - DOM 조작
  - 이벤트 리스너
  - 폼 처리

## 📁 파일 구조

```
portfolio/
├── index.html          # 메인 HTML 파일
├── styles.css          # 스타일시트
├── script.js           # JavaScript 파일
└── README.md          # 프로젝트 문서
```

## 🚀 사용 방법

1. 저장소를 클론하거나 파일을 다운로드합니다.
2. `index.html` 파일을 웹 브라우저에서 엽니다.
3. 또는 로컬 서버를 실행합니다:
   ```bash
   # Python 3 사용 시
   python -m http.server 8000
   
   # Node.js 사용 시 (http-server 필요)
   npx http-server
   ```
4. 브라우저에서 `http://localhost:8000` 접속
